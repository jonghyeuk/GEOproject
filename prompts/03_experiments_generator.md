# Experiments 카테고리 문서 생성기

> 상위 규칙: 반드시 `00_global_rules.md`를 먼저 읽고 따를 것

## 역할
Atlas Experiments 카테고리의 문서를 생성합니다.
실험 설계, 변인 통제, 데이터 수집, 안전 수칙 관련 문서를 다룹니다.

## 입력 정보
- 실험 주제 또는 방법론 주제
- 대상 학년
- 난이도

## 출력 JSON 필드
- docType: `"article"` 또는 `"guide"`
- categoryId: `"experiments"`

## 본문 구조 (body)
```
## 정의
(이 실험/방법이 무엇인지)

## 이 실험으로 무엇을 확인할 수 있는가
(실험의 목적과 가치)

## 핵심 개념
### 변인 설정
- 독립변인: ...
- 종속변인: ...
- 통제변인: ...

### 실험 절차
1. ...
2. ...

## 데이터 수집 방법
- 측정 도구
- 기록 방식
- 반복 횟수

## 자주 하는 실수
- ...

## 안전 수칙
- ...

## 결과 해석 방법
- ...
```

## 주의사항
- 안전 수칙을 반드시 포함 ("지도교사 감독 하에" 문구 필수)
- 변인 통제의 구체적 예시 포함
- "왜 이렇게 해야 하는지" 이유 설명
