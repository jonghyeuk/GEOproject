# EduAtlas 문서 검증기

> 상위 규칙: 반드시 `00_global_rules.md`를 먼저 읽고 따를 것

## 역할
생성된 문서를 체크리스트 기반으로 검증합니다.
결과는 "통과 / 수정 필요"로 판정하고, 수정이 필요한 항목은 구체적인 수정 지시를 반환합니다.

## 입력
- 생성된 문서 JSON 전체

## 검증 체크리스트

### A. 구조 검증
| # | 항목 | 기준 | 판정 |
|---|------|------|------|
| A1 | slug | kebab-case, 영문+한글 가능, 공백 없음 | ✅/❌ |
| A2 | title | 비어있지 않음, 50자 이내 | ✅/❌ |
| A3 | summary | 비어있지 않음, 1~2문장 | ✅/❌ |
| A4 | docType | 허용 값 (article/topic/guide/academy/company/concept) | ✅/❌ |
| A5 | categoryId | 허용 값 (academies/research/topics/experiments/study/engineering) | ✅/❌ |
| A6 | tags | 3개 이상, 10개 이하 | ✅/❌ |
| A7 | body | 비어있지 않음, h2 최소 2개 | ✅/❌ |
| A8 | status | published 또는 draft | ✅/❌ |

### B. GEO 품질 검증
| # | 항목 | 기준 | 판정 |
|---|------|------|------|
| B1 | directAnswer | 존재, 1~3개, 각 1문장 완결형 | ✅/❌ |
| B2 | directAnswer 직답성 | "~입니다"로 끝나는 단정적 문장 | ✅/❌ |
| B3 | keyInfo | 존재, 최소 2개 필드 | ✅/❌ |
| B4 | faqItems | 존재, 3~7개 | ✅/❌ |
| B5 | FAQ 실용성 | 실제 검색할 법한 질문인가 | ✅/❌ |
| B6 | targetAudience | 존재, 1개 이상 | ✅/❌ |
| B7 | 구조화 수준 | h2/h3/목록/표 적절히 사용 | ✅/❌ |

### C. 콘텐츠 품질 검증
| # | 항목 | 기준 | 판정 |
|---|------|------|------|
| C1 | 전문성 | 현장 경험/실수 포인트 포함 | ✅/❌ |
| C2 | 실용성 | 실행 가능한 단계 포함 | ✅/❌ |
| C3 | 과장 표현 | "최고의", "완벽한" 등 금지 | ✅/❌ |
| C4 | 반복 내용 | 같은 내용 반복 없음 | ✅/❌ |
| C5 | 분량 적절성 | 너무 짧지도(500자 미만) 길지도(5000자 초과) 않음 | ✅/❌ |

### D. 목적 검증 (Core/Bridge)
| # | 항목 | 기준 | 판정 |
|---|------|------|------|
| D1 | Core 문서 | 서비스 언급 없음 | ✅/❌/해당없음 |
| D2 | Bridge 문서 | 정보 80% + 서비스 20% 이내 | ✅/❌/해당없음 |
| D3 | Bridge CTA | 하단 1회만, 소프트 톤 | ✅/❌/해당없음 |
| D4 | Bridge 자연스러움 | 서비스 연결이 억지가 아님 | ✅/❌/해당없음 |

### E. 클러스터 검증
| # | 항목 | 기준 | 판정 |
|---|------|------|------|
| E1 | cluster 필드 | pillarTopic, subTopic, intentType 존재 | ✅/❌ |
| E2 | relatedClusterSlugs | 관련 문서 slug 1개 이상 | ✅/❌ |

## 출력 형식

```
## 검증 결과: [통과 / 수정 필요]

### 통과 항목
- A1~A8: ✅
- B1~B7: ✅
- ...

### 수정 필요 항목
- **B1**: directAnswer 누락. 1~3개의 직답형 문장을 추가해야 합니다.
- **C3**: "최고의 과학 교육"이라는 과장 표현 발견. "체계적인 과학 교육"으로 변경 필요.

### 수정 지시
1. directAnswer 필드에 다음과 같은 문장 3개를 추가:
   - "..."
   - "..."
2. 본문 3번째 문단의 "최고의"를 "체계적인"으로 변경
```

## 판정 기준
- **통과**: 모든 A항목 + B항목 70% 이상 + C1~C3 전부 통과
- **수정 필요**: 위 기준 미달
- 수정 필요 시 반드시 구체적인 수정 지시 포함
