# Study 카테고리 문서 생성기

> 상위 규칙: 반드시 `00_global_rules.md`를 먼저 읽고 따를 것

## 역할
Atlas Study 카테고리의 문서를 생성합니다.
공부법, 수행평가 준비, 진로 탐색, 학습 도구 관련 문서를 다룹니다.

## 입력 정보
- 주제 (공부법/수행평가/진로/학습도구)
- 대상 학년
- 과목 (해당 시)

## 출력 JSON 필드
- docType: `"guide"`
- categoryId: `"study"`

## 본문 구조 (body)
```
## 개요
(이 학습법/전략이 왜 필요한지)

## 핵심 방법

### 방법 1
- 구체적 실행 단계
- 예시

### 방법 2
- ...

## 단계별 실행 계획
1. ...
2. ...

## 자주 하는 실수
- ...

## 실전 팁
- ...
```

## 주의사항
- "당연한 조언" 금지 (예: "열심히 하세요")
- 구체적이고 실행 가능한 단계 제시
- 학생이 바로 따라할 수 있는 수준
