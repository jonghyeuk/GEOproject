# Interactive 카테고리 문서 생성기

> 상위 규칙: 반드시 `00_global_rules.md`를 먼저 읽고 따를 것

## 역할
Atlas Interactive 카테고리의 문서를 생성합니다.
인터렉티브 과학 시뮬레이션에서 추출한 교육 콘텐츠를 구조화된 문서로 변환합니다.

## 문서 생성 워크플로우

### 시뮬레이션 → 문서 추출 프로세스
```
1. 인터렉티브 시뮬레이션 HTML 분석
2. 핵심 과학 원리 추출 → Core 문서 생성
3. 실험 방법론 추출 → Experiments 연결 문서
4. 키워드/태그 도출 → 기존 문서와 연결
5. FAQ 및 퀴즈 데이터 활용 → directAnswer/faqItems 생성
```

## 입력 정보
- 시뮬레이션 HTML 파일
- 과학 분야 (물리/화학/생물/지구과학/융합)
- 대상 학년 수준

## 출력 JSON 필드
- docType: `"article"` 또는 `"concept"` 또는 `"guide"`
- categoryId: `"interactive"`
- `interactiveDemo`: 연결된 시뮬레이션 슬러그 (예: `"beer-lambert-law"`)

## 본문 구조 (body)
```
## 핵심 원리
(시뮬레이션에서 다루는 과학 법칙/원리 설명)

## 실험 개요
### 목적
### 준비물
### 실험 절차

## 시뮬레이터로 체험하기
(인터렉티브 데모 링크 및 사용법)

## 실생활 적용
- ...

## 심화 학습
- ...
```

## directAnswer 예시
```json
[
  "{법칙명}은(는) ...을 설명하는 법칙입니다.",
  "이 시뮬레이션에서 ...를 직접 변화시키며 ...를 확인할 수 있습니다."
]
```

## 추출 대상 키워드 유형
- 과학 법칙명 (예: Beer-Lambert 법칙)
- 핵심 변수 (예: 흡광도, 투과율, 농도)
- 실험 도구 (예: 프리즘, 분광기)
- 교과 연결 (예: 화학I, 물리II)

## 연결 문서 자동 생성
하나의 시뮬레이션에서 최대 3가지 유형의 문서를 추출:
1. **개념 문서** (concept) → 핵심 과학 원리 설명
2. **실험 가이드** (guide) → 실험 절차 + 보고서 양식
3. **주제 문서** (topic) → 탐구 주제로서의 확장 가능성

## 주의사항
- 시뮬레이션의 이론 탭 내용을 그대로 복사하지 말 것
- 교과과정 연결을 반드시 포함
- 안전 수칙이 있다면 반드시 포함
- interactiveDemo 필드로 시뮬레이션과 양방향 연결
