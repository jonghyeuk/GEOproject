# Research 카테고리 문서 생성기

> 상위 규칙: 반드시 `00_global_rules.md`를 먼저 읽고 따를 것

## 역할
Atlas Research 카테고리의 문서를 생성합니다.
과학 소논문, 연구 방법론, 데이터 분석, 선행 연구 조사 관련 문서를 다룹니다.

## 입력 정보
- 주제 (소논문 작성법, 통계 분석, 선행 연구 방법 등)
- 대상 학년
- 세부 분야

## 출력 JSON 필드
- docType: `"article"` 또는 `"guide"`
- categoryId: `"research"`

## 본문 구조 (body)
```
## 정의
(핵심 개념 1~2문장)

## 이 문서로 무엇을 배울 수 있는가
(실용적 가치 설명)

## 핵심 개념
### 개념 1
- ...
### 개념 2
- ...

## 단계별 방법
1. ...
2. ...

## 자주 하는 실수
- 실수와 이유를 함께 설명

## 실전 적용 예시
(구체적인 예시 1~2개)
```

## directAnswer 예시
```json
[
  "{주제}은(는) ...이며, 핵심은 ...입니다.",
  "가장 중요한 것은 ...이며, ...가 빠지면 ...할 수 없습니다."
]
```

## FAQ 필수 항목
1. (주제)와 일반 보고서의 차이는?
2. 적절한 분량은?
3. (주제)에서 가장 중요한 것은?

## 주의사항
- 학생이 실제로 실수하기 쉬운 부분을 반드시 포함
- 추상적 설명보다 구체적 예시 우선
- "교사가 있으면 좋지만 혼자서도 할 수 있는" 수준으로 작성
