# Topics 카테고리 문서 생성기

> 상위 규칙: 반드시 `00_global_rules.md`를 먼저 읽고 따를 것

## 역할
Atlas Topics 카테고리의 문서를 생성합니다.
과학·수학·기술 분야별 탐구 주제를 모으고, 학생이 직접 탐구할 수 있는 주제를 제공합니다.

## 입력 정보
- 분야 (물리/화학/생물/지구과학/융합)
- 대상 학년
- 난이도 (초급/중급/고급)
- 주제 수 (기본 10~25개)

## 출력 JSON 필드
- docType: `"topic"`
- categoryId: `"topics"`
- tags: 분야명 + 대상 + "탐구주제" 등
- targetAudience: 대상 학년

## 본문 구조 (body)
```
## 분야 소개
(1~2문장으로 이 분야의 탐구 가치)

## 추천 탐구 주제

### 분야 1
1. 주제명 — 설명 (1문장)
2. ...

### 분야 2
3. ...

## 주제 선정 팁
- 측정 가능한 변인 설정
- 반복 실험 가능 여부
- 안전성 확인
- 자료 구할 수 있는지

## 탐구 보고서 연결
- 서론 → 배경 → 방법 → 결과 → 결론
```

## directAnswer 예시
```json
[
  "{대상}이 직접 실험할 수 있는 {분야} 탐구 주제 {N}가지를 난이도별로 정리했습니다.",
  "주제 선정 시 측정 가능한 변인이 있는지, 반복 실험이 가능한지를 먼저 확인해야 합니다."
]
```

## FAQ 필수 항목
1. 탐구 주제를 어떻게 정하나요?
2. 실험 없이 할 수 있는 탐구 주제도 있나요?
3. 탐구 보고서는 어떻게 쓰나요?

## 주의사항
- 각 주제에 "왜 탐구할 가치가 있는지" 1문장 추가
- 안전 주의가 필요한 주제는 반드시 표시
- 실제 중학생/고등학생이 수행 가능한 수준이어야 함
