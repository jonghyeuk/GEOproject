# Engineering 카테고리 문서 생성기

> 상위 규칙: 반드시 `00_global_rules.md`를 먼저 읽고 따를 것

## 역할
Atlas Engineering 카테고리의 문서를 생성합니다.
반도체, 공정, 장비, 기술 트렌드 관련 문서를 다룹니다.

## 입력 정보
- 기술/공정명
- 세부 분야 (반도체/공정/장비/시뮬레이터)
- 수준 (입문/중급/전문)

## 출력 JSON 필드
- docType: `"article"` 또는 `"concept"`
- categoryId: `"engineering"`

## 본문 구조 (body)
```
## 정의
(기술/공정의 핵심 정의 1~2문장)

## 핵심 개념
### 원리
- ...
### 종류/분류
- ...

## 공정 단계 (해당 시)
### 1. ...
### 2. ...

## 적용 분야
- ...

## 최신 기술 동향
- ...

## 관련 기술
- ...
```

## directAnswer 예시
```json
[
  "{기술명}은(는) ...하는 공정/기술입니다.",
  "핵심 원리는 ...이며, 주로 ...에 사용됩니다."
]
```

## 주의사항
- 전문 용어는 영문 병기 (예: 식각(Etching))
- 최신 기술 동향 포함 (연도 명시)
- 관련 기술과의 연계성 설명
