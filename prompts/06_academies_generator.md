# Academies 카테고리 문서 생성기

> 상위 규칙: 반드시 `00_global_rules.md`를 먼저 읽고 따를 것

## 역할
Atlas Academies 카테고리의 문서를 생성합니다.
교육 기관, 학원, AI 교육, 교육 방법 관련 문서를 다룹니다.

## 입력 정보
- 기관명 또는 교육 주제
- 지역 (해당 시)
- 대상
- 교육 과목
- 교육 특징

## 출력 JSON 필드
- docType: `"academy"` (기관) 또는 `"article"` (교육 방법론)
- categoryId: `"academies"`

## 기관 문서 본문 구조
```
## 기관 소개
(개요 1~2문장)

## 교육 프로그램
### 주요 과목
- ...
### 특징
- ...

## 교육 방식
- ...

## 위치 및 연락처
- ...
```

## 교육 방법론 문서 본문 구조
```
## 정의
(교육 방법/트렌드의 핵심 정의)

## 주요 내용
### ...
- ...

## 교사를 위한 활용 방법
- ...

## 학생 관점의 장점
- ...

## 주의사항
- ...
```

## keyInfo (기관 문서)
```json
{
  "name": "기관명",
  "location": "지역",
  "subjects": "주요 과목"
}
```

## 주의사항
- 기관 문서는 자동 게시 금지 (운영자 검수 필수)
- 허위 정보, 과대 광고 표현 금지
- EducationalOrganization schema 자동 적용됨
