화학 물리

IDEAL GAS LAW

이상기체 법칙 인터랙티브 시뮬레이터
PV = nRT

01 — 기체란 무엇인가?

우리가 숨 쉬는 공기, 풍선 안의 헬륨, 타이어를 채우는 질소… 이것들은 모두 기체(Gas)입니다.

기체 분자들은 끊임없이 움직이고, 서로 부딪히고, 벽에 충돌합니다. 이 충돌이 바로 압력(Pressure)의 정체입니다.

분자들은 더 빠르게 움직일수록(온도↑) 벽을 더 세게 치고, 공간이 좁을수록(부피↓) 더 자주 부딪힙니다.

02 — 세 가지 변수의 춤

이상기체 법칙의 핵심은 네 가지 변수의 관계입니다:

PV = nRT

여기서 R은 기체 상수(8.314 J/mol·K = 0.08206 L·atm/mol·K)로, 자연이 정해준 고정 상수입니다.

P, V, T, n 중 일부를 고정하면 나머지 변수들의 관계가 드러납니다.

02.5 — n은 무엇인가? 몰(mol)의 정체

화학에서 n(몰, mol)은 물질의 양을 세는 단위입니다. 연필을 "다스(12개)"로 세듯, 분자는 몰로 셉니다.

🔸 1몰 = 6.022 × 10²³ 개

이 숫자를 아보가드로 수(Nₐ)라고 부릅니다.

🔸 몰 질량 — 원소마다 1몰의 무게가 다르다

수소(H₂) 1몰 = 2g, 산소(O₂) 1몰 = 32g, 물(H₂O) 1몰 = 18g

🔸 PV=nRT에서 n의 역할

n이 커지면(기체가 많아지면) 같은 온도·부피에서 압력이 비례해서 증가합니다.

n = PV / RT   |   n = 질량(g) / 몰 질량(g/mol)

03 — 고정 조건별 법칙

🔸 압력 고정 (등압 과정 — 샤를의 법칙)

풍선을 냉동실에 넣으면 쪼그라듭니다.

V₁/T₁ = V₂/T₂  (P 일정)

🔸 부피 고정 (등적 과정 — 게이-뤼삭 법칙)

밀봉된 캔을 불에 가열하면 폭발합니다.

P₁/T₁ = P₂/T₂  (V 일정)

🔸 온도 고정 (등온 과정 — 보일의 법칙)

주사기의 끝을 막고 피스톤을 누르면 압력이 올라갑니다.

P₁V₁ = P₂V₂  (T 일정)

💻 아두이노 코드

#include <Wire.h>
#include <Adafruit_BMP280.h>
#include <OneWire.h>
#include <DallasTemperature.h>

Adafruit_BMP280 bmp;
OneWire oneWire(2);
DallasTemperature ds(&oneWire);

void setup() {
  Serial.begin(9600);
  bmp.begin(0x76);
  ds.begin();
  Serial.println("P(hPa),T_bmp(C),T_ds(C)");
}

void loop() {
  ds.requestTemperatures();
  float P = bmp.readPressure() / 100.0;  // hPa
  float T1 = bmp.readTemperature();       // °C
  float T2 = ds.getTempCByIndex(0);       // °C
  Serial.print(P); Serial.print(",");
  Serial.print(T1); Serial.print(",");
  Serial.println(T2);
  delay(1000);
}

📋 실험 방법

1

장치 조립

BMP280과 DS18B20 센서를 아두이노에 연결하고, 주사기에 실리콘 튜브로 밀폐 연결합니다.

2

초기 상태 기록

시리얼 모니터를 열고 현재 기압(P₀)과 온도(T₀)를 기록합니다. 주사기 부피(V₀)도 확인합니다.

3

등온 실험 (보일의 법칙)

온도를 일정하게 유지하며 주사기 피스톤을 눌러 부피를 변경합니다. 각 부피에서 압력을 기록합니다.

💡 P₁V₁ = P₂V₂가 성립하는지 확인!
4

등적 실험 (게이-뤼삭 법칙)

주사기를 고정(부피 일정)한 채 드라이어로 가열합니다. 온도 변화에 따른 압력 변화를 기록합니다.

💡 P₁/T₁ = P₂/T₂가 성립하는지 확인!
5

데이터 수집

각 실험에서 최소 5개 데이터 포인트를 수집합니다. 시리얼 모니터의 값을 결과보고 탭에 입력합니다.

6

분석 및 보고

결과보고 탭에서 PV=nRT로 계산한 이론값과 실험값을 비교합니다.
